# Log4A
Android simple "LogCat" library to improve debugging

## Install

Use Gradle via Maven:
```Java
repositories {
    maven {
        url 'https://dl.bintray.com/kito/maven/'
    }
}
```

Gradle:
```Java
compile 'com.devkito.log4a:log4a:1.1.0'
```
